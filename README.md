# Pyxel-1942
This project is a modern take on the classic retro plane game *1942*. It is built using the Pyxel game library, and it features a fully playable environment where you control a plane in dynamic combat. Each entity in the game, from enemies to player controls, is managed through individual files for easy customization and readability.

---

Key highlights:

* Custom-built game board for an immersive experience
* All assets were designed using Pyxel's built-in engine
* Modular file structure for each game entity
* Main file to seamlessly launch and start gameplay
